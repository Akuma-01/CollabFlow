# CollabFlow

Academic project collaboration platform for student teams and faculty mentors.

## Tech Stack
- Node.js + Express 5
- PostgreSQL
- JWT Authentication

## Setup
1. Clone the repo
2. Copy `backend/.env.example` to `backend/.env` and fill in values
3. Run `npm install` inside `backend/`
4. Run `node server.js`


## Future Features
- Project showcase/discovery page with upvoting
- Public project profiles
- University-wide project leaderboard
- Project cover images/GIFs via file upload
- Domain-based access control: users with verified college email 
  domains can view projects without explicit invitation
